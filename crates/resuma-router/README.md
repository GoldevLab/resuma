# resuma-router

File-based routing helpers for [Resuma](https://github.com/GolfredoPerezFernandez/resuma).

Scans `src/pages/` and maps files like `[id].rs` to routes such as `/users/:id`.

**Docs:** [docs.rs/resuma-router](https://docs.rs/resuma-router) · **Main crate:** [resuma](https://crates.io/crates/resuma)
