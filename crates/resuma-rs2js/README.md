# resuma-rs2js

Rust → JavaScript compiler for [Resuma](https://github.com/GolfredoPerezFernandez/resuma) event handlers.

Translates a safe subset of Rust expressions into JavaScript so click handlers resume lazily in the browser without hydration.

**Docs:** [docs.rs/resuma-rs2js](https://docs.rs/resuma-rs2js) · **Main crate:** [resuma](https://crates.io/crates/resuma)
